[2.8.0] - 2026-06-21
---

### 新功能

* **自定义再分期** — 提前还款支持自定义还款年限 / 月供（issue #5）。
  * 「提前还款 / 再分期」面板新增"自定义目标"：可按目标剩余期数或目标月供重算还款计划，提前还款金额改为选填（支持不还钱、纯重新分期）。
  * 等额本息支持按期数和按目标月供（按目标月供估算可缩短到的期数，实际月供约为该值）；等额本金仅支持按期数。
  * 还款模拟「一次性还款」新增"缩短至期数 / 提高月供"两种处理方式。
  * 新增 `estimateReamortize` 纯函数，表单实时预估与提交计算同源，保证所见即所得。

[2.7.0] - 2026-04-05
---

### 新功能

* **组合贷款** — 支持将商业贷款 + 公积金贷款组合为一组，合并计算合计月供/利息/本金。
  * LoanGroup 数据类型、CombinedLoanHelper 合并计算、组合管理（创建/删除）
  * 计算页合计展示 + 独立/合计视图切换（CombinedViewTabs）
  * 导入导出支持组合方案，合并表导出适配
  * 分析页变更影响叠加展示，对比页适配组合方案（当期指标、利率拆分）
  * 还贷模拟支持组合视图，提前还款优化分配算法
* **机会成本增强** — 回本周期计算、流动性提示、投资对比曲线、关键节点竖线标记。

### 修复

* 修复组合方案对比数据 + 增加当期指标和利率拆分。
* 修复智能分析多个计算问题 + UI 优化。
* 修复机会成本分析路由逻辑、观察期和投资收益计算。
* 修复组合贷款合计月供计算、还款模拟组合视图交互、提前还款优化分配算法。
* PWA 更新提示显示实际新版本号而非当前版本。

[2.6.0] - 2026-04-02
---

### 新功能

* [New] 还款模拟页（/simulate）— 支持调整月供和一次性提前还款两种模式，实时对比原方案与模拟方案的利息、期数、月供变化。
* [New] 机会成本分析 — 基于理财年化收益率计算提前还款的机会成本，支持自定义观察期（截止日期/快捷选择），展示净收益对比。
* [New] 智能分析（测试中）— 自动采样不同时间点和金额组合，生成全局最优、性价比最优、轻松方案三类推荐；可选扣除机会成本和通胀折现。
* [New] 模拟对比图表 — 原方案 vs 模拟方案的月供/剩余本金双轴折线图，支持点击移动模拟起点。
* [New] InfoTip 组件 — 结果卡片和机会成本指标添加解释图标，基于 shadcn/ui Tooltip 封装。

### 改进

* 月供调整改为输入新月还款额（绝对值），支持增减双向调整。
* 提前还款支持缩短期限和减少月供两种策略，智能分析同步适配。
* 理财收益率输入支持小数，新增预设按钮快捷选择。
* 观察期支持截止日期和 1年/2年 快捷选择，按天精确计算。
* 执行日期支持期数+日期双输入，金额精确到两位小数。

### 修复

* 修复智能分析忽略 lumpSumStrategy、小额贷款采样步长为 0、边际拐点回退次优、缩短期数 -1 等多项缺陷。
* 修复减少月供时观察期对比数据不显示、机会成本分析方向错误。
* 修复观察期过滤排除无日期期数，减少月供不延长贷款期限。
* 统一所有页面标题和边距，暗色模式按钮样式统一。

[2.0.0] - 2026-03-29
---

### 全量重构

从 vanilla TypeScript 全量迁移到 React 19 + Shadcn/ui + Tailwind CSS 4 + Zustand 技术栈。
包管理从 npm 切换到 pnpm。

### 全新 UI

* Dashboard 风格布局，桌面端侧边导航 + 移动端底部 Tab。
* Shadcn/ui (Radix UI) 组件库，无障碍支持。
* 亮色/暗色/跟随系统三种主题模式。
* 响应式设计适配手机、平板、桌面。

### 新功能

* [New] 数据分析页面 — 还款概览环形图、变更前后对比线图、利息节省柱图、详细趋势图。
* [New] 贷款方案管理 — 支持保存、加载、重命名、删除多个贷款方案。
* [New] LPR 内置利率表 — 5 年期 LPR 历史数据（2019-08 至 2026-03），支持基点偏移自动计算。
* [New] 自定义利率表 — 通用利率时间线管理器，预留多数据源扩展。
* [New] 数据持久化 — localStorage 自动保存/恢复，刷新不丢失数据。
* [New] 还款计划虚拟滚动 — 360 期数据流畅滚动（@tanstack/react-virtual）。
* [New] 还款趋势面积图 — 本金/利息占比可视化，变更点竖线标注（ECharts）。
* [New] Excel 导出保留。

### 架构

* `core/` 纯计算逻辑与 UI 框架解耦，可跨平台复用。
* `features/` 按功能分区，模块化卡片设计，新增功能 = 新增目录。
* Zustand 状态管理替代事件订阅模型，persist 中间件对接 localStorage。
* StorageAdapter 接口抽象存储层，未来可替换为 IndexedDB 或云端同步。
* RateProvider 接口预留 LPR 等多数据源扩展。
* vite-plugin-pwa 自动 Service Worker 注册。

### 工程化

* pnpm 严格依赖模式。
* Vitest + @testing-library/react + jsdom 测试。
* CI 新增 PR check workflow（type-check + lint + test + build）。
* Biome 2 支持 JSX/TSX。
